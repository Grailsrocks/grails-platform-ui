APIs and Tags for Theming and Plugin UI integration

(Quick Reference)

grailsUISets Bean

The grailsUISets bean is an instance of the UISets class provided by Platform UI.

You can call methods on this bean to set the current request's UI Set, discover available UI Sets and so on.

See the UISets docs for details.
